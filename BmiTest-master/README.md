# Java EE tutorial examples

##檔案結構

src/ -- 包含Java一般的Class和Servlet

WebContent/ -- JSP放在這

WebContent/WEB-INF/lib -- 外部jar放這


## Build the examples

BmiA.java -- 單純用Servlet實現Bmi

BmiB.java -- 使用Servlet傳參數給Jsp顯示Bmi

HelloWorld.java -- 使用單一JSP實現Bmi

TestProject -- 利用Google搜尋結果顯示搜尋頁面
